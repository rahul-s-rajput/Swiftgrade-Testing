Q1. Differentiate f(x) = 3x³ - 2x² + 5x - 7 [10 marks]
Correct Answer:
f'(x) = 9x² - 4x + 5
Step-by-step Solution:
1. Apply power rule: d/dx(xⁿ) = n·xⁿ⁻¹
2. f'(x) = d/dx(3x³) - d/dx(2x²) + d/dx(5x) - d/dx(7)
3. f'(x) = 3(3x²) - 2(2x) + 5(1) - 0
4. f'(x) = 9x² - 4x + 5
Marking Scheme:
• Correct application of power rule: 3 marks
• Derivative of 3x³ → 9x²: 2 marks
• Derivative of -2x² → -4x: 2 marks
• Derivative of 5x → 5: 2 marks
• Derivative of constant → 0: 1 mark
Total: 10 marks
Q2. Solve the integral: ∫(2x² + 3x - 1)dx [15 marks]
Correct Answer:
(2x³/3) + (3x²/2) - x + C
Step-by-step Solution:
1. Separate the integral: ∫2x²dx + ∫3xdx - ∫1dx
2. Apply power rule for integration: ∫xⁿdx = xⁿ⁺¹/(n+1) + C
3. ∫2x²dx = 2(x³/3) = 2x³/3
4. ∫3xdx = 3(x²/2) = 3x²/2
5. ∫1dx = x
6. Combine: (2x³/3) + (3x²/2) - x + C
Marking Scheme:
• Separating the integral correctly: 2 marks
• Integration of 2x²: 4 marks
• Integration of 3x: 4 marks
• Integration of -1: 3 marks
• Including constant of integration C: 2 marks
Total: 15 marks

Q4. Calculate area of circle with radius r = 7.5 cm [8 marks]
Correct Answer:
Area = 176.71 cm² (or 56.25π cm²)
Step-by-step Solution:
1. Formula: A = πr²
2. Substitute r = 7.5: A = π(7.5)²
3. Calculate: A = π × 56.25
4. A = 3.14159 × 56.25 ≈ 176.71 cm²
Marking Scheme:
• Stating correct formula: 2 marks
• Correct substitution: 2 marks
• Calculation of 7.5²: 2 marks
• Final answer with units: 2 marks
Total: 8 marks

Q6. Ball thrown upward with v₀ = 20 m/s. Find max height and time [15 marks]
Correct Answer:
Maximum height: 20.4 m
Time to reach max height: 2.04 s
Step-by-step Solution:
1. At max height, v = 0
2. Using v = v₀ + gt: 0 = 20 + (-9.8)t
3. Solve for t: t = 20/9.8 = 2.04 s
4. Using h = v₀t + ½gt²:
5. h = 20(2.04) + ½(-9.8)(2.04)²
6. h = 40.8 - 20.4 = 20.4 m
Marking Scheme:
• Identifying v = 0 at max height: 2 marks
• Correct kinematic equation for time: 3 marks
• Calculating time: 3 marks
• Correct kinematic equation for height: 3 marks
• Calculating height: 3 marks
• Correct units: 1 mark
Total: 15 marks

Q8. Find roots of x² - 5x + 6 = 0 [8 marks]
Correct Answer:
x = 2 or x = 3
Method 1 - Factoring:
1. x² - 5x + 6 = 0
2. Find factors of 6 that add to -5: -2 and -3
3. (x - 2)(x - 3) = 0
4. x = 2 or x = 3

Method 2 - Quadratic Formula:
1. x = [-(-5) ± √(25-24)]/2
2. x = (5 ± 1)/2
3. x = 3 or x = 2
Marking Scheme:
• Correct factorization or formula setup: 3 marks
• Finding first root: 2 marks
• Finding second root: 2 marks
• Verification (optional but good): 1 mark
Total: 8 marks

Q9. Convert 75°F to Celsius and Kelvin [7 marks]
Correct Answer:
75°F = 23.89°C = 297.04 K
Step-by-step Solution:
1. Formula: °C = (°F - 32) × 5/9
2. °C = (75 - 32) × 5/9 = 43 × 5/9 = 23.89°C
3. Formula: K = °C + 273.15
4. K = 23.89 + 273.15 = 297.04 K
Marking Scheme:
• Correct Fahrenheit to Celsius formula: 2 marks
• Calculation to Celsius: 2 marks
• Correct Celsius to Kelvin formula: 1 mark
• Calculation to Kelvin: 2 marks
Total: 7 marks
